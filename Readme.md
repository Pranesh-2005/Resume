## ğŸš€ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pranesh-2005/Resume.git
   cd Resume
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view your Resume.

## ğŸ’» Usage

- **Customize your information:**  
  Edit the configuration files in `components/Resume/Resume.config.tsx` to add your name, skills, experience, and project details.
- **Update styles:**  
  Change theme colors and fonts in the styled-components files (e.g., `PortfolioLayout.styles.tsx` and `Resume.styles.tsx`).
- **Add new sections:**  
  Use or extend the modular components in the `components/` directory to further personalize your resume.
- **Deploy:**  
  You can easily deploy your site to [Vercel](https://vercel.com/), [Netlify](https://www.netlify.com/), or any static hosting provider.
